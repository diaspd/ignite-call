## 🚀 Getting started

Clone the project and access the folder.

```bash
$ git clone https://github.com/diaspd/
$ cd 
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run dev
```

The app will be available on `http://localhost:3000`
